A) Ecriture du code HTML pour définir le contenu de votre page

Un fichier HTML, sert à creer la base d'un site WEB, il se compose généralement de 3 grande partie, l'entête (head) ou on décrit les fichiers (invisible depuis le navigateur), le corps (body) ou on écrit le contenu du site (visible), et le pieds de page (footer). Une balise est un élément de texte (un nom) encadrée par le caractère inférieur (<) et le caractère supérieur (>). par exemple « <H1> ».

On peu inserer un commentaire avec la balise <!-- "commentaire" -->

Dans le menu de navigation on utilise une liste, delimitée par les balises "ul" et chaques points de cette liste sera défini par une balise "li". Il y a aussi un "href" qui sert pour les liens (interne au site, ou externe).

Les balises de titre "hx" (x est un nombre entre 1 et 6) servent à donner une certaine importance à un bout de texte. Quand aux balises "p" elles servent à delimitter un paragraphe, et pour un saut de ligne, on utilise une balise particulière, la balise <br/> qui s'utilise toute seule.
On peut aussi ajouter des images avec la balise "img" avec un src="chemin de l'image" sous cette forme:
" <img src="adresse de l'image"/> " on peut ajouetr une infobulle en mettant " title="contenu infobulle" " dans la balise image.
